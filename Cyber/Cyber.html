Kali Linux


ls

  1. ls
Permet de voir le contenu (lister) du dossier/répertoire courant
Option utile : -a, --all : pour affichez tout (notament les fichiers cachées) -A, --almost-all : pour presque affichez tout (notament les fichiers cachées sans affichez . et .. qui représent respactivement le répertoire courant et le répertoire parent ) -A, --help : pour affichez la documentation de base de la commande -R, --recursive : pour affichez récursivement le contenu du répertoire courant et des sous-répertoires -s, --size : pour affichez la taille des fichier Utilisation : ls
[OPTION]...
[FICHIER]... Example : Le fichier peut être un répertoire : ls .. ls sl ls / -R -a ls . -a ls -a .
